Javascript er programmerbar kode som blant annet kan brukes i HMTL dokumenter. 
JavaScript kan brukes til å forandre/manipulere en nettside,
enten det er snakk om et bilde eller å printe ut informasjon via for eksempel en knapp.

1) 

Forskjell på id og class

CSS id:
Id'er er unike, hver side kan kun ha ett elemt med en type id.

CSS Class: 
Man kan bruke samme class på flere elementer, man kan bruke flere typer class på samme element.

2)

Vurder og sammenlign brukerbarhet til versjoner i deloppgave 0 og deloppgave 1:

Når det gjelder brukbarhet og design så ser versjonen i deloppgave 1 bedre ut på en nettside fordi den bruker
mindre plass. Den ser også bedre ut. Den er mer brukervennlig; lettere å velge og se igjennom
de forskjellige programmeringsspråk, den er mer oversiktlig. På den andre siden kan
det være lettere å finne frem til ønsket programmeringsspråk i deloppgave 0 ved å bruke
ctrl + f for å gjøre ett søk i nettleser(spesielt hvis det er snakk om over 100 elementer).

3)

Wikipedia løser problemstillingen med en lang liste av programmeringsspråk ved å bruke
forbokstaven til de ulike programmeringsspråkene. Slik at hvert programmeringsspråk som
begynner på A, B eller C osv er i hver sin gruppe. Eks A: A+, A++ B: BASIC, osv. 

4) 
Den mest merkbare forskjellen mellom et div og span element, er hvordan elementene vises.
Div er et block element, mens span er et inline element. En div block viser en seksjon av
et dokument på siden. Et span element inneholder deler av informasjon på linje med innhold
i nærheten.

5) Brukbarhetsproblemer i deloppgave 2:
Når man har valgt et språk, kommer det kun opp en linje som sier at et språk er valgt.
Ingen funskjon utover dette. Når man trykker på et programmeringsspråk kunne man for
eksempel hatt litt mer informasjon om språket man har valgt, enn kun "Du har valgt språket".
Dette kan løses ved å skrive i JavaScript filen under e_outputbox.innerHTML. Man kunne også
hatt med en overskrift som kommer opp for hvert språk som blir valgt.
