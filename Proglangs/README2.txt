Kontrollspørsmål

Forklar hvilken deler av din applikasjon så langt, implementerer de forskjellige lagene i MVC
arkitekturen. MVC står får Model­View­Controller, som til norsk kan oversettes til
Modell­Presentasjon­Kontorller.

MVC (Model - view - controller) har tre filer: HTML, CSS og JavaScript. 
Model er HTML filen, siden denne filen er modellen og holder på dataen vår. 
View er CSS filen, som er den er filen som bestemmer hvordan dataen vår skal vises. 
Controller er JavaScript fila. Det er fila som skal kontrollere hva som skal vises av informasjon i HTML fila(Model).
